GameplayTag

概念

Gameplay标签（分层标签系统） 是用户定义的字符串，充当概念性的分层标签。你可以将Gameplay标签应用于项目中的对象，并对其求值以驱动你的Gameplay实现，类似于检查布尔值或标记。Gameplay标签本质上是一种有层级结构的文本标识，通常以 “.” 作为层级分隔符，即x.y.z
例如eg.Player.Attack和eg.Player.Attack.Melee，其中近战Melee就是攻击方式的一种子类型

应用场景

场景 示例
输入系统 InputTag.Move → 触发移动逻辑
技能系统 Ability.Fire → 播放开火动画 + 生成子弹
状态机 State.Crouching + State.Moving → 切换到蹲走动画
Buff/Debuff Effect.Burning → 每秒扣血
UI 显示 有 State.LockedOn 标签 → 显示锁定图标
AI 决策 敌人有 State.Stunned → 不攻击
