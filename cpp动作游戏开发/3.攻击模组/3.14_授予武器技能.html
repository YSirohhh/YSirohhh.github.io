  4. 在收起武器后，我们需要移除装备武器后的技能授权
    回到WarriorGameplayAbility查看移除技能的EndAbility函数

    每执行一次授予技能的函数GiveAbility(Spec)就会返回一个该技能所对应的Handle，可以说，Spec就是一个人，而Handle就是Spec的身份证，并且Spec为技能实例的一部分
    现在我们就要用OutGrantedAbilitySpecHandles存储GiveAbility(Spec)返回的Handle

    编译之后回到unreal会发现在授予技能的函数那多了一个输出引脚OutGrantedAbilitySpecHandles这就是所有武器技能的Handle，将来用于清除技能

C++ 参数写法 蓝图引脚位置 UHT 的理解
float Value 左侧 (Input) 传入一个数值。
const FString& Name 左侧 (Input) 传入一个大的字符串（只读）。
FVector& OutLocation 右侧 (Output) 函数计算出一个位置并带出来。
TArray& MyArray 右侧 (Output) 函数会填充这个数组并带出来。
UPARAM(ref) TArray& MyArray 左侧 (Input) 传入一个数组并在函数内直接修改它。
